Variables and Data Types

- Convention camelCase. var [name] = [value]

- Data Types

PrimitiveNumber, String, Boolean, Undefined, Null và Symbol (ES6)

ComplexFunction và Object

Function: var f = function() {...statements [;]...}: Chỉ được thực thi khi gọi f()

Object: var o = {...key: value [,]...}: value có thể có kiểu dữ liệu bất kì

Array: var a = [...item [,]...]: item có thể có kiểu dữ liệu bất kì

Array là một Object với key là các chỉ số bắt đầu từ 0. Ta có thể kiểm tra qua method Array.isArray()

- Check Data Type Sử dụng typeof [var]

- Special Values

undefined: Một biến chỉ được khai báo mà không gán giá trị ngay

null: Một biến được gán giá trị null. Mặc định có kiểu Object

NaN: Một biến Number nhưng kết quả không xác định. VD: 100 / 'ABC'. Ta có thể kiểm tra qua hàm boolean isNaN(value)

Infinity: Một biến Number nhưng kết quả vô cùng. VD: 100 / 0

Some Built-in Functions

- Alert Hiển thị hộp thoại thông báo. Chỉ có nút OK

- Confirm Hiển thị hộp thoại thông báo. Có nút OK và Cancel

boolean confirm(string message);

- Prompt Hiển thị hộp thoại thông báo. Có nút OK và Cancel. Có thêm EditText Box

string prompt(string message);

- Console Quản lý Tab console của Inspect Tools

void console.log('[message]'); hoặc console.log([variable]);

- Set Timeout Thực hiện một function sau một khoảng thời gian

setTimeout(function() {...}, [time_in_ms])

- Set Interval Thực hiện một function liên tục cứ sau một khoảng thời gian

Some special

- Exponent (**). 2**3 = 8

- Type Comparison (=== và !==). So sánh cả về giá trị và kiểu dữ liệu. VD: 1 == '1' : true, nhưng 1 === '1' : false

- And Working (&&). Giá trị trả về là giá trị false (false, null, undefined, NaN, '', 0) đầu tiên hoặc mặc định trả về giá trị cuối cùng. VD: 1 && 'A' trả về 'A'

- Or Working (||). Giá trị trả về là giá trị true đầu tiên hoặc mặc định trả về giá trị cuối cùng. VD: NaN || 'A' || 1 trả về 'A'

- For In Duyệt các Key của một Object (bao gồm cả Array và String): for(var key in o) { }

- For Of Duyệt các Value của một Array hoặc String hoặc Iterable Object: for(var value of a) { }

- Some String methods

FindTìm vị trí xuất hiện đầu tiên của một chuỗi hoặc -1, bắt đầu tìm kiếm tại một vị trí xác định: .indexOf(string, start_index)

VD: console.log('Hello World'.indexOf('l', 4)); : 9

Find LastTương tự Find nhưng hướng tìm kiếm từ cuối lên đầu: .lastIndexOf(string, start_index)

VD: console.log('Hello World'.lastIndexOf('l', 4)); : 3

Find using REGEXTìm vị trí xuất hiện đầu tiên theo REGEX: .search(regex)

Get SubstringLấy một chuỗi con tại vị trí xác định, chỉ số âm thể hiện chiều ngược lại (bắt đầu từ -1): .slice(start_index, end_index)

VD: console.log('Hello World'.slice(6, 10)); : Worl

VD: console.log('Hello World'.slice(6, -2)); : Wor

ReplaceThay thế các kết quả tìm kiếm một chuỗi bằng một chuỗi khác. Nếu dùng string trực tiếp sẽ chỉ thay thế chuỗi đầu tiên tìm thấy: .replace(string, start_index)

VD: console.log('Hello World'.replace('ll', 'r')); : Hero World

Case ConvertingChuyển đổi về UpperCase hoặc LowerCase: .toUpperCase() và .toLowerCase()

VD: console.log('Hello World'.toUpperCase()); : HELLO WORLD

TrimXóa bỏ khoảng trắng và các kí tự kết thúc dòng ở đầu và/hoặc cuối chuỗi: .trim() hoặc .trimStart() hoặc .trimEnd()

VD: console.log('Hello World'.indexOf('l', 4)); : 9

SplitTách các phần của một chuỗi sử dụng một delimiter và chứa kết quả vào mảng: .split(delimiter, count) với count xác định số lượng chuỗi tách được sẽ được đưa vào mảng kết quả

VD: console.log('JavaScript, PHP, Ruby'.split(', ', 2)); : ['JavaScript', 'PHP']

JoinNối các phần tử của mảng sử dụng một delimiter và trả về một String : .join(delimiter)

PushThêm toàn bộ một hoặc nhiều phần tử vào cuối mảng và trả về chiều dài của mảng sau khi thêm: .push(...items)

VD: console.log([123, {name: "Vũ Xuân Bắc", age: 22}, 'Hello World'].push(null, undefined, false)); : 6 : [123, {…}, 'Hello World', null, undefined, false]

UnshiftThêm toàn bộ một hoặc nhiều phần tử vào đầu mảng và trả về chiều dài của mảng sau khi thêm: .push(...items)

VD: console.log([123, {name: "Vũ Xuân Bắc", age: 22}, 'Hello World'].unshift(null, undefined, false)); : 6 : [null, undefined, false, 123, {…}, 'Hello World']

ConcatThêm một hoặc nhiều phần tử hoặc một mảng khác vào cuối mảng và trả về mảng mới chứa kết quả sau khi thêm: .concat(...items) hoặc .concat(array)

VD: console.log([123, {name: "Vũ Xuân Bắc", age: 22}, 'Hello World'].concat([null, undefined, false])); : [123, {…}, 'Hello World', null, undefined, false]

SplicingXóa một hoặc nhiều phần tử bắt đầu từ vị trí xác định và thêm một hoặc nhiều phần tử vào vị trí bị xóa, trả về một mảng chứa các phần tử bị xóa: .splice(start, count,...items)

VD: console.log([123, {name: "Vũ Xuân Bắc", age: 22}, 'Hello World'].splice(1, 2, null, undefined, false)); : [{...}, 'Hello World'] : [123, null, undefined, false]

SlicingCắt một hoặc nhiều phần tử bắt đầu từ vị trí xác định, kết thúc tại vị trí xác định và trả về mảng mới chứa kết quả: .splice(start, count,...items)

VD: console.log([123, {name: "Vũ Xuân Bắc", age: 22}, 'Hello World'].slice(1, 2)); : [{...}]

- Some Array methods using Iterating

For EachDuyệt các phần tử của mảng và thực hiện một hàm với mỗi phần tử: .forEach(function(value, index){ })

VD: console.log(['Hello World', 123, {name: "Vũ Xuân Bắc", age: 22}, null].forEach(function(value, index){ console.log(`${index}: ${value}`)} ));

Is True For AllKiểm tra xem tất cả các phần tử của mảng thỏa mãn một điều kiện cho trước xác định bởi một hàm: .every(function(value, index){ })

VD: console.log(['Hello World', 123, {name: "Vũ Xuân Bắc", age: 22}].every(function(value, index){ return value !== null } )); : true

Has Any TrueKiểm tra xem có phần tử nào của mảng thỏa mãn một điều kiện cho trước xác định bởi một hàm: .some(function(value, index){ })

VD: console.log(['Hello World', 123, {name: "Vũ Xuân Bắc", age: 22}].some(function(value, index){ return value === 123; } )); : true

Find ItemTìm phần tử đầu tiên của mảng thỏa mãn một điều kiện cho trước xác định bởi một hàm: .find(function(value, index){ })

VD: console.log(['Hello World', 123, {name: "Vũ Xuân Bắc", age: 22}].find(function(value, index){ return value > 0} )); : 123

Find AllTìm tất cả các phần tử của mảng thỏa mãn một điều kiện cho trước xác định bởi một hàm: .filter(function(value, index){ })

VD: console.log(['Hello World', 123, {name: "Vũ Xuân Bắc", age: 22}].filter(function(value, index){ return value !== null } ));: ['Hello World', 123, {...}]

MapThay thế tất cả các phần tử của mảng bằng giá trị trả về của một hàm và trả về mảng mới: .map(function(value, index, originArray){ })

VD: console.log(['Hello World', 123, {name: "Vũ Xuân Bắc", age: 22}].map(function(value, index){ return value.toString().toUpperCase() } ));: ['HELLO WORLD', '123', '[OBJECT OBJECT]']

ReduceThực hiện một tính toán trên tất cả các phần tử của mảng thông qua một hàm và lưu trữ (thông qua giá trị trả về của hàm) nó lại sử dụng cho phần tử phía sau: .reduce(function(accumulate, value, index, originArray){ }, initial_value_for_accumulate)

VD: console.log(['Hello World', 123, {name: "Vũ Xuân Bắc", age: 22}].reduce(function(accumulate, value, index){ return accumulate + ":" + value.toString() }, 0 ));: '0:Hello World:123:[object Object]'

Working on Objects

- Object Creation

Key của Object có thể nhận bằng giá trị của một biến bằng việc thêm [] cho Key đó.

VD: var key = 123; var o = { [key]: 'A number' }; console.log(o) : { 123: 'A number' }

Ta có thể truy xuất (Get, Set) value của một key hoặc thêm một key bằng .key hoặc ['key']

VD: var key = 123; var o = { [key]: 'A number' }; o.name = 'My Object'; console.log(o['123']); : 'A number' : { 123: 'A number', name: 'My Object'}

Ta có thể xóa một key bằng delete object.key

- Constructor Khai báo như một Function trong đó sử dụng từ khóa this để xác định các Keys cho object.

VD: function User(name, avatar){ this.name = name; this.avatar = avatar }

var user1 = new User('Vũ Xuân Bắc', link_to_avatar)

Function này sẽ được gán cho __proto__ constructor của Object.

VD: user1.constructor === User

Một Object được tạo qua Constructor có thể có thêm các Key-Value khác

VD: user1.age = 12 : { name: 'Vũ Xuân Bắc', avatar: ...., age: 12 }

- Prototype Nguyên mẫu chung cho các Object. Khi thay đổi Prototype thì các Object sẽ thay đổi theo.

Prototype chỉ được dùng với Class. Thay đổi Prototype là thay đổi ở mức Class. Giống với static members trong Class.

Truy cập vào Prototype của một Class bằng thuộc tính prototype

VD: function User(name, avatar){ this.name = name; this.avatar = avatar }

User.prototype.new_key = new_value

Functions

- Creation

Declaration FunctionSử dụng từ khóa function trước tên hàm: function [name]([params]) { [statements;] }

Expression Function (Anonymous Function) Sử dụng từ khóa function và không có tên hàm: function([params]) { [statements;] }. Ta cũng có thể đặt tên cho hàm với cú pháp tương tự Declaration Function

Arrow FunctionKhông cần sử dụng từ khóa function và cũng không cần chỉ định tên: ([params]) => { [statements;] }. Nếu hàm chỉ có lệnh trả về thì có thể bỏ {} và "return". Chú ý là Arrow Functions không có context, tức là không thể sử dụng từ khóa this bên trong nó

Điểm khác biệt giữa Declaration Function và Expression Function là ở cơ chế Hosting khi mà Declaration Function cho phép gọi trước khi định nghĩa, còn Expression Function thì không

Khi có các hàm trùng tên thì hàm được định nghĩa sau sẽ ghi đè.

Các tham số không chỉ định kiểu dữ liệu

- Calling

Một hàm chỉ được định nghĩa thì không thể thực thi. Ta có thể gọi tên hàm để thực thi một hàm

Khi truyền thiếu đối số thì các đối số còn lại mặc định nhận giá trị undefined

Một hàm có thể nhận số lượng đối số bất kì. Tất cả các đối số truyền vào hàm được lưu vào Object arguments, kể cả với các tham số có tên và không tên. Ta có thể truy xuất các phần tử trong arguments qua vòng lặp for...of

Một hàm không return sẽ mặc định trả về undefined
